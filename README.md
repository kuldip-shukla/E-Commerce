# E-Commerce
 Demo e-commerce site example with admin panel it is like OLX where seller can sell his products and buyer can only buy seller's product.

### Prerequisites
```
git clone https://github.com/kuldip-shukla/E-Commerce.git
npm install
ng serve
```
